# Sample Library Upload Guide

1. Put redacted/watermarked sample files into `public/samples/`.
2. Supported formats: `pdf`, `png`, `jpg`, `jpeg`, `webp`.
3. You can create subfolders by country or type, for example:
   - `public/samples/singapore/`
   - `public/samples/malaysia/`
   - `public/samples/company/`
4. File names are displayed as titles on `/en/samples` and `/zh/samples`.

Notes:
- Do not upload any unredacted personal information.
- The page is public and intended for format reference only.
